# I2C Scanner for Arduino

Простой сканер I2C‑устройств для Arduino. Определяет адреса всех подключённых к шине I2C устройств.

## Особенности

- Автоматическое сканирование диапазона адресов `0x01`–`0x7F`
- Вывод найденных устройств с форматированием (ведущий ноль для адресов < 0x10)
- Обработка ошибок передачи
- Периодическое сканирование (каждые 5 с)

## Как использовать

1. Загрузите скетч на плату Arduino.
2. Откройте Serial Monitor (`Ctrl+Shift+M` в Arduino IDE).
3. Установите скорость 115200 бод.
4. Наблюдайте результаты сканирования в консоли.
